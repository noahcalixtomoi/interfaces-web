## 🚀 Meus Focos de Estudo & Aplicação

* **Mobile-First & Responsividade:** Interfaces que se adaptam perfeitamente a qualquer tamanho de tela.
* **Acessibilidade (a11y):** Garantir que a web seja para todo mundo, seguindo as melhores práticas do W3C.
* **Performance:** Código limpo e otimizado para que as páginas carreguem num piscar de olhos.
